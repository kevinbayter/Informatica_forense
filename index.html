Informática Forense - Eje 3

By Kevin Bayter

Versión Lanzamiento Interfaz Compatibilidad Redes Sistema de archivos
Windows 95 Agosto de 1995 Introdujo el menú Inicio Soporte para IBM PC Soporte "Plug and Play" FAT32
Lanzamiento: Windows 95 fue lanzado el 24 de agosto de 1995 y se considera una de las versiones más significativas de Windows debido a los grandes cambios que introdujo.
Interfaz de Usuario: Fue la primera versión que introdujo el menú Inicio, la barra de tareas y el botón de inicio. La interfaz fue completamente rediseñada para ser más intuitiva.
Compatibilidad y Hardware: Ofreció soporte mejorado para computadoras IBM PC y soporte para "Plug and Play".
Redes y Conectividad: Introdujo el soporte para Microsoft Network y proporcionó integración con Internet, incluido un navegador web temprano llamado Internet Explorer.
Sistema de Archivos: Introdujo el sistema de archivos FAT32, que permitía el uso de discos duros más grandes.
Multitarea: Mejoró la gestión de las aplicaciones en ejecución.
Juegos: Incluyó juegos clásicos como Minesweeper, Solitaire y Hearts.
Soporte para 32 bits: Fue el paso inicial hacia un sistema operativo completamente de 32 bits.
Duración del soporte: Microsoft finalizó el soporte oficial el 31 de diciembre de 2001.
Windows 98 Junio de 1998 Mejoras en el menú Inicio Mejor soporte para ACPI y dispositivos USB Mejoras en la conectividad a Internet FAT32
Lanzamiento: Windows 98 se lanzó el 25 de junio de 1998. Está basado en el núcleo Windows 95 y se le considera una versión de transición, con mejoras significativas en comparación con su predecesor.
Interfaz de Usuario: Incorporó el "Explorador de Windows" que combinaba la funcionalidad del administrador de archivos y el programa de lanzamiento. También introdujo el "Escritorio Web" que permitía a los objetos Active Desktop ser colocados en el fondo del escritorio.
Compatibilidad y Hardware: Añadió soporte para numerosos estándares de dispositivos, incluidos USB, AGP y ACPI. Además, ofreció mejoras en el rendimiento general del sistema.
Redes y Conectividad: Windows 98 fue el primero en incluir Internet Explorer 4 y sus tecnologías relacionadas como Outlook Express, Windows Address Book y NetMeeting. También introdujo el cliente de red para Microsoft Networks y el Asistente para la conexión a Internet.
Sistema de Archivos: Continuó con el soporte de FAT32, pero añadió la utilidad de conversión Drive Converter para convertir unidades FAT16 a FAT32 sin perder datos.
Multimedia: Incorporó el soporte para DVD y la capacidad de abrir y cerrar la bandeja del CD-ROM mediante software.
Actualizaciones: En 1999, se lanzó Windows 98 Second Edition (SE) que ofrecía mejoras en la velocidad, confiabilidad y seguridad sobre la versión original.
Duración del soporte: Microsoft finalizó el soporte oficial para Windows 98 el 11 de julio de 2006.
Windows ME (Millennium Edition) Septiembre del 2000 Introdujo el "Restaurar sistema" Eliminación del modo de operación en tiempo real Sistema de Compartición de Internet FAT32
Lanzamiento: Windows ME (Millennium Edition) se lanzó el 14 de septiembre de 2000. Fue la última versión de Windows basada en el núcleo de Windows 95.
Interfaz de Usuario: Introdujo características como Restaurar sistema, para revertir la configuración del sistema a una fecha anterior, y la Galería de imágenes y videos. También vio la eliminación del Modo MS-DOS real.
Compatibilidad y Hardware: Aunque estaba basado en el núcleo de Windows 95, eliminó el modo de operación en tiempo real, lo que afectó la compatibilidad con algunos programas más antiguos. Sin embargo, introdujo características como la Protección de archivos del sistema para prevenir la corrupción accidental de archivos críticos del sistema.
Redes y Conectividad: Introdujo el Sistema de Compartición de Internet, que permitía a múltiples computadoras en una red local compartir una única conexión a Internet. También vino con Internet Explorer 5.5 y Windows Media Player 7.
Sistema de Archivos: Continuó con el soporte de FAT32, pero no ofreció soporte nativo para NTFS.
Multimedia: Mejoras en la reproducción de medios y la introducción de Windows Movie Maker.
Actualizaciones: A pesar de introducir varias características nuevas, Windows ME es a menudo considerado como uno de los lanzamientos menos estables de Microsoft.
Duración del soporte: Microsoft finalizó el soporte oficial para Windows ME el 11 de julio de 2006, junto con Windows 98.
Windows 2000 Febrero del 2000 Basado en el núcleo de Windows NT Mejoras en la seguridad y estabilidad Introdujo Active Directory NTFS 3.0
Lanzamiento: Windows 2000 fue lanzado el 17 de febrero de 2000. Aunque sigue la numeración de Windows 95/98, en realidad está basado en el núcleo de Windows NT y fue diseñado para ser una actualización de Windows NT 4.0.
Interfaz de Usuario: Mientras que mantuvo una apariencia similar a versiones anteriores, introdujo numerosas mejoras en la interfaz de usuario, y fue el primero en incluir la "Ayuda y soporte técnico" de Microsoft.
Compatibilidad y Hardware: Diseñado tanto para estaciones de trabajo como para servidores, ofreció un equilibrio entre compatibilidad y rendimiento. Introdujo muchas de las tecnologías de "Plug and Play" presentes en Windows 95/98 en el núcleo de Windows NT.
Redes y Conectividad: Introdujo Active Directory, que es una mejora significativa en la gestión de redes sobre los dominios en Windows NT. También ofreció mejoras en los protocolos de red y en la conectividad de red en general.
Sistema de Archivos: Introdujo la versión 3.0 de NTFS, que ofrecía numerosas mejoras sobre las versiones anteriores, incluyendo la capacidad de cifrar archivos y carpetas para mejorar la seguridad.
Seguridad: Mejoras en la seguridad del sistema operativo, incluyendo la introducción de las políticas de seguridad y la capacidad de establecer permisos a nivel de archivo y carpeta.
Modos de Operación: A diferencia de Windows ME, Windows 2000 mantuvo el modo de operación en tiempo real, permitiendo una mejor compatibilidad con aplicaciones más antiguas.
Duración del soporte: Microsoft finalizó el soporte oficial para Windows 2000 el 13 de julio de 2010.
Windows XP Octubre del 2001 Nueva interfaz llamada Luna Soporte para aplicaciones más recientes Firewall de Windows y conectividad inalámbrica Soporte mejorado para NTFS
Lanzamiento: Windows XP fue lanzado el 25 de octubre de 2001. Es conocido por su estabilidad y eficiencia en comparación con versiones anteriores.
Interfaz de Usuario: Introdujo una nueva interfaz llamada Luna, que ofrecía un diseño más limpio y moderno en comparación con versiones anteriores. También introdujo el "Centro de Ayuda y Soporte".
Compatibilidad y Hardware: Ofreció un soporte mejorado para hardware y software más reciente, incluidos los juegos. Incluía un modo de compatibilidad para ayudar a ejecutar programas más antiguos.
Redes y Conectividad: Mejoras significativas en la conectividad inalámbrica, introducción del Firewall de Windows, y la capacidad de configurar redes domésticas más fácilmente.
Sistema de Archivos: Aunque NTFS no era nuevo, Windows XP ofreció mejoras en su rendimiento y seguridad, incluyendo la capacidad de mover archivos mientras estaban en uso.
Seguridad: A pesar de que se introdujeron numerosas mejoras de seguridad, Windows XP también fue objeto de numerosas vulnerabilidades, lo que llevó a Microsoft a lanzar el Service Pack 2 en 2004, que centraba en mejoras de seguridad.
Ediciones: Windows XP estaba disponible en varias ediciones, incluyendo Home, Professional, y más tarde, ediciones Media Center y Tablet PC.
Duración del soporte: Microsoft finalizó el soporte oficial para Windows XP el 8 de abril de 2014, aunque algunas organizaciones continuaron recibiendo actualizaciones de seguridad bajo acuerdos personalizados.
Windows Vista Enero del 2007 Interfaz Aero con efectos de transparencia Mejoras en seguridad con UAC Mejoras en conectividad inalámbrica y seguridad de red Soporte continuado para NTFS
Lanzamiento: Windows Vista se lanzó el 30 de enero de 2007 y se considera uno de los lanzamientos más polémicos de Microsoft debido a los problemas iniciales de rendimiento y compatibilidad.
Interfaz de Usuario: Introdujo la interfaz Aero, que ofrecía efectos de transparencia y animaciones más fluidas. También incluyó características como Flip 3D y la barra lateral con gadgets.
Compatibilidad y Hardware: A pesar de las mejoras visuales, muchos usuarios tuvieron problemas iniciales con la compatibilidad de hardware y software, especialmente en sistemas más antiguos.
Redes y Conectividad: Presentó una nueva versión del Internet Explorer, el IE7, y mejoras significativas en la seguridad de red y la conectividad inalámbrica.
Sistema de Archivos: Continuó con el soporte de NTFS y también introdujo características como ReadyBoost y ReadyDrive.
Seguridad: Introdujo el Control de Cuentas de Usuario (UAC) para mejorar la seguridad, aunque fue considerado molesto por muchos usuarios.
Ediciones: Estaba disponible en varias ediciones, desde Home Basic hasta Ultimate, cada una con diferentes conjuntos de características.
Duración del soporte: Microsoft finalizó el soporte oficial para Windows Vista el 11 de abril de 2017.
Windows 7 Octubre del 2009 Mejoras en la barra de tareas y bibliotecas Mejoras en rendimiento y compatibilidad Centro de redes y recursos compartidos Soporte continuado para NTFS y mejoras en rendimiento
Lanzamiento: Windows 7 se lanzó el 22 de octubre de 2009 y fue bien recibido por los usuarios y críticos, siendo considerado una mejora significativa sobre Windows Vista.
Interfaz de Usuario: Introdujo una barra de tareas renovada con una mejor funcionalidad de "anclaje" y las bibliotecas en el Explorador de Windows.
Compatibilidad y Hardware: Mejoró el rendimiento y la compatibilidad con aplicaciones y dispositivos en comparación con Vista. También introdujo el modo Windows XP para ejecutar aplicaciones más antiguas.
Redes y Conectividad: Introdujo el "Centro de redes y recursos compartidos" y mejoras en la conectividad inalámbrica.
Sistema de Archivos: Continuó con el soporte de NTFS y mejoras en el rendimiento de acceso a discos y archivos.
Seguridad: Mejoras en el firewall de Windows y en Windows Defender. También introdujo BitLocker To Go para el cifrado de dispositivos extraíbles.
Ediciones: Al igual que Vista, estaba disponible en varias ediciones, desde Starter hasta Ultimate.
Duración del soporte: Microsoft finalizó el soporte oficial para Windows 7 el 14 de enero de 2020.
Windows 8 Octubre del 2012 Interfaz Modern UI (anteriormente Metro) Optimizado para pantallas táctiles Introducción de la tienda Windows Store Soporte mejorado para NTFS y sistema de archivos ReFS
Lanzamiento: Windows 8 fue lanzado el 26 de octubre de 2012. Representó un cambio significativo en el diseño en comparación con versiones anteriores, con un enfoque en dispositivos táctiles.
Interfaz de Usuario: Introdujo la interfaz Modern UI (anteriormente conocida como Metro) con baldosas en vivo y la eliminación del tradicional menú Inicio. Diseñado para ser amigable con pantallas táctiles.
Compatibilidad y Hardware: Diseñado para funcionar tanto en PCs tradicionales como en tablets, con optimizaciones específicas para dispositivos táctiles.
Redes y Conectividad: Introducción de la tienda Windows Store, permitiendo a los usuarios descargar aplicaciones diseñadas específicamente para la interfaz Modern UI.
Sistema de Archivos: Aunque NTFS seguía siendo el sistema de archivos principal, Windows 8 introdujo el sistema de archivos Resilient File System (ReFS) diseñado para ser más resistente a las corrupciones de datos.
Seguridad: Introdujo características como Windows Defender con capacidades completas de antivirus y Secure Boot para mejorar la seguridad durante el arranque del sistema.
Ediciones: Estaba disponible en varias ediciones, incluyendo Windows 8, Windows 8 Pro y Windows 8 RT (para dispositivos ARM).
Duración del soporte: El soporte principal finalizó el 9 de enero de 2018, mientras que el soporte extendido está programado para finalizar en 2023.
Windows 8.1 Octubre del 2013 Mejoras en la interfaz y reintroducción del botón Inicio Optimizaciones y mejoras en la experiencia táctil Mejoras en la tienda Windows Store NTFS y mejoras en ReFS
Lanzamiento: Windows 8.1 fue lanzado el 17 de octubre de 2013 como una actualización gratuita para usuarios de Windows 8. Abordó muchas de las críticas dirigidas a Windows 8.
Interfaz de Usuario: Aunque mantuvo la interfaz Modern UI, reintrodujo el botón Inicio y ofreció más opciones de personalización. También permitió a los usuarios iniciar directamente en el escritorio tradicional.
Compatibilidad y Hardware: Introdujo mejoras en la gestión de múltiples monitores y optimizaciones adicionales para dispositivos táctiles.
Redes y Conectividad: La tienda Windows Store recibió una revisión significativa, con un diseño mejorado y una mayor selección de aplicaciones.
Sistema de Archivos: Continuó con el soporte de NTFS y presentó mejoras en el sistema de archivos ReFS.
Seguridad: Mejoras en la seguridad de Internet Explorer y mayor integración con servicios en la nube de Microsoft, como OneDrive.
Ediciones: Similar a Windows 8, estaba disponible en varias ediciones, incluyendo Windows 8.1, Windows 8.1 Pro y Windows 8.1 RT.
Duración del soporte: El soporte principal finalizó el 9 de enero de 2018, mientras que el soporte extendido está programado para finalizar en 2023, al igual que Windows 8.
Windows 10 Julio del 2015 Interfaz híbrida con Menú Inicio expandido Unificado para PC, dispositivos móviles y Xbox Introducción de Microsoft Edge y Cortana Soporte para NTFS y mejoras en ReFS
Lanzamiento: Windows 10 fue lanzado el 29 de julio de 2015 y es la versión actual principal de Windows en el momento de mi última actualización (2021).
Interfaz de Usuario: Presenta una interfaz híbrida que combina elementos de Windows 7 y Windows 8.1, con un Menú Inicio expandido que incluye baldosas en vivo. Se realizan actualizaciones y refinamientos constantes en la interfaz a través de actualizaciones semestrales.
Compatibilidad y Hardware: Diseñado como un sistema operativo unificado para PC, dispositivos móviles, Xbox, y otros dispositivos. También introdujo el modo Windows 10 S, una versión más ligera y segura del sistema operativo.
Redes y Conectividad: Introdujo el navegador Microsoft Edge como reemplazo de Internet Explorer. También vino con la asistente virtual Cortana integrada y mejoras en la tienda Windows Store, ahora simplemente llamada Microsoft Store.
Sistema de Archivos: Continuó con el soporte de NTFS y presentó mejoras en el sistema de archivos ReFS, especialmente en servidores y estaciones de trabajo.
Seguridad: Introdujo Windows Hello, que permite a los usuarios iniciar sesión con reconocimiento facial, huellas dactilares o un PIN. También mejoró las capacidades de BitLocker y introdujo Windows Defender Antivirus y Windows Defender Firewall como soluciones de seguridad integradas.
Ediciones: Disponible en varias ediciones, desde Home y Pro hasta versiones empresariales y educativas.
Actualizaciones: Windows 10 introdujo el concepto de "Windows como servicio", donde las actualizaciones se entregan de manera continua en lugar de lanzar nuevas versiones del sistema operativo.
